# Image Assets Required

Place the following image files in this folder (`assets/images/`):

| File               | Used In Scene                         |
|--------------------|---------------------------------------|
| campus.jpg         | Scene 0 (morning wake-up), Scene 2   |
| orientation.jpg    | Scene 1 (arrived early), Scene 3     |
| classroom.jpg      | Scene 4, 5, 6 (classroom moments)    |
| orgfair.jpg        | Scene 7 (org fair + friends)         |
| homework.jpg       | Scene 8 (quiet night, homework)      |
| canteen.jpg        | Scene 9 (canteen social lunch)       |
| ending_bad.jpg     | Scene 10 – Bad Ending                |
| ending_good.jpg    | Scene 11 – Good Ending               |
| ending_neutral.jpg | Scene 12 – Neutral Ending            |

## Tips
- Recommended size: 800×450 px (16:9 landscape)
- Format: JPG or PNG (rename to .jpg if needed)
- Free sources: Unsplash (unsplash.com), Pexels (pexels.com)
- Search terms: "college campus", "classroom students", "school canteen", "university orientation"

> Note: If an image file is missing the app will show a school icon placeholder
> automatically — the app will NOT crash.

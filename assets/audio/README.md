# Audio Assets Required

Place the following audio files in this folder (`assets/audio/`):

| File         | When It Plays                          |
|--------------|----------------------------------------|
| click.mp3    | Every time a choice button is tapped   |
| success.mp3  | When the Good Ending is reached        |
| fail.mp3     | When the Bad Ending is reached         |
| neutral.mp3  | When the Neutral Ending is reached     |

## Tips
- Keep files short: click ~0.2s, endings ~2–3s
- Format: MP3 (or OGG — rename to .mp3)
- Free sources: Freesound (freesound.org), Mixkit (mixkit.co)
- Search: "button click sound", "success fanfare", "fail sound effect"

> Note: If an audio file is missing the app will continue silently — it will NOT crash.
